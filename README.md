# Local RAG Workspace

Start the local services:

```bash
docker compose up -d --build
```

New users create an account at [http://127.0.0.1:8100/register](http://127.0.0.1:8100/register). All users then log in and use the RAG workspace at [http://127.0.0.1:8101](http://127.0.0.1:8101).

After login, choose **Account** to view the signed-in email. Use the Chainlit user menu to log out.
